# ReadMe — MS4 Orquestador

Orquestador de solicitudes de compra/intercambio de libros de ReadMe.

### Desarrollo local

**Variables de entorno**

```bash
cp .env.example .env
```

Edita `.env` si necesitas cambiar credenciales, puertos o URLs de los otros microservicios.

**Correr la app**

```bash
npm install
npm start
```

La API queda disponible en `http://localhost:8004`.
Swagger UI en `http://localhost:8004/swagger`.
